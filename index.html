Git examples

    # create a new repo in the current directory
    $ git init

    # add a file to the staging area
    $ git add <file>

    # show current state of stage
    $ git status

    # commit changes to the local branch
    $ git commit -m "message"

    # point local repository at remote origin
    $ git remote add origin https://github.com/me/project.git

    # ruin a coworker's day
    $ git push origin master

GitHub

  • A for-profit "social network" for git users
  • Public repos for free, private for pay
  • Free hosting for small web projects
  • Convention for repo naming is "kebab-case"

HTML

  • HyperText Markup Language
    • Text not constrained to be linear
  • Not a programming language
  • Defines the structure of a web page
    • Text is "marked up" to define how it should be displayed
    • For example, <p> represents a paragraph
  • Invented by Tim Berners Lee, first published in 1991
  • HTML 5 was introduced in 2014
HTML 5 is defined by:
  • HTML: structure
  • CSS: appearance
  • JavaScript: functionality
Example: 
    <tagname attribute="value">content</tagname>

Atom
