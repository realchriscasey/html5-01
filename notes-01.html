HTML

  • HyperText Markup Language
    • Text not constrained to be linear
  • Not a programming language
  • Defines the structure of a web page
    • Text is "marked up" to define how it should be displayed
    • For example, <p> represents a paragraph
  • Invented by Tim Berners Lee, first published in 1991
  • HTML 5 was introduced in 2014
HTML 5 is defined by:
  1. HTML: structure
  2. CSS: appearance
  3. JavaScript: functionality
Example: 
    <tagname attribute="value">content</tagname>

Semantic Elements

Semantic Elements are meaningful tags that have an explicit effect on the way a page's content is rendered by the browser. Their names are human-understandable and they have similar effects on all compliant web browsers. Using appropriate tags helps screen readers and other accessibility tools better understand the page's structure.

block outline

Block level semantic elements define a separate group of content. These groups are rendered as different visual elements. Some examples include <p>, <pre> and <article>

inline

Inline semantic elements affect text within a line. They don't cause a line break and don't indicate that the text is a different set of context. Some examples include <b>, <i> and <span>
