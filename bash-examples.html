Counting CSS lines

For Assignment 3, one requirement is to write 50 lines of CSS.

$ grep -v ^\s*$ styles/styles.css | wc -l
27
$

Huzzah! We've counted only the non-empty lines in our CSS. Only 27? Looks like we need to get back to work...
